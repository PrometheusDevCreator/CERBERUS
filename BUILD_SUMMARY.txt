================================================================================
ARCHITECTURE HIGHLIGHTS
================================================================================

Event System:
  - StandardEventEnvelope with v0 system contract
  - ID prefixes: evt_, ses_, thr_, msg_
  - Sequence-ordered, session-scoped events
  - JSON payload with message content, artifacts, metadata

Message Flow:
  1. User sends via WebSocket
  2. Server creates message.created event, persists to DB
  3. Event broadcasts to all connected clients
  4. Server calls agent API(s) based on routing target
  5. Agent response becomes new event
  6. Cycle repeats in real-time

Database:
  - PostgreSQL with asyncpg pool
  - Single events table with 20 columns
  - Indexes on (session_id, sequence) and (session_id, thread_id, sequence)
  - JSON payload/artifacts/meta stored as JSONB
  - Auto-migration on startup

Agents:
  - call_sarah() → OpenAI v1/chat/completions
    Model: gpt-5.4 (configurable)
    Timeout: 120s, Max tokens: 4096
  
  - call_claude() → Anthropic v1/messages
    Model: claude-opus-4-6 (configurable)
    Timeout: 120s, Max tokens: 4096

WebSocket Protocol:
  - Client sends: {"command": "send_message", "session_id": "...", 
                   "thread_id": "thr_main", "payload": {...}}
  - Server broadcasts: EventEnvelope as JSON
  - Bi-directional, no authentication (MVP)

================================================================================
DEPLOYMENT QUICKSTART
================================================================================

Local Development:
  1. pip install -r requirements.txt
  2. cp .env.example .env && edit with your API keys & local DB URL
  3. python -m uvicorn app.main:app --reload --port 8000
  4. Visit http://localhost:8000

Railway Deployment:
  1. railway init (select "Empty")
  2. railway add (select PostgreSQL)
  3. railway variables set (OPENAI_API_KEY, ANTHROPIC_API_KEY, etc)
  4. railway up (or connect GitHub repo for auto-deploy)
  5. App available at: https://your-project.railway.app

Environment Variables Required:
  - OPENAI_API_KEY
  - ANTHROPIC_API_KEY
  - SARAH_MODEL (default: gpt-5.4)
  - CLAUDE_MODEL (default: claude-opus-4-6)
  - DATABASE_URL
  - PORT (default: 8000)
